描述

本接口用于获取征信信息。

接口地址

http://www.hn9xiang.com/api/query

支持格式

JSON

请求方式

GET

请求示例

http://www.hn9xiang.com/api/query?name=张三&mobile=18888888888&idcard=580580199808080882

请求参数

参数名称 参数类型 数据类型 必填 说明
name query string 是 姓名
mobile query string 是 手机号
idcard query string 是 身份证号

调用示例

  PHP
  1. <?php
  2. $curl = curl_init();
  3. curl_setopt_array($curl, array(
  4.   CURLOPT_URL => "http://www.hn9xiang.com/api/query?name=张三&mobile=18888888888&idcard=580580199808080882",
  5.   CURLOPT_RETURNTRANSFER => true,
  6.   CURLOPT_ENCODING => "",
  7.   CURLOPT_MAXREDIRS => 10,
  8.   CURLOPT_TIMEOUT => 30,
  9.   CURLOPT_HTTP_VERSION => CURL_HTTP_VERSION_1_1,
  10.   CURLOPT_CUSTOMREQUEST => "GET",
  11.   CURLOPT_HTTPHEADER => array(
  12.     "accept: application/json",
  13.     "api-key: 您的api-key",
  14.     "content-type: application/json"
  15.   ),
  16. ));
  17. $response = curl_exec($curl);$err = curl_error($curl);
  18. curl_close($curl);
  19. if ($err) {
  20.   echo "cURL Error #:" . $err;
  21. } else {
  22.   echo $response;
  23. }
  24.
